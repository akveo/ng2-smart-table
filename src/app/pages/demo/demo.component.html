安装

该库可用作npm软件包，因此您只需运行以下命令即可：（如果不是新手，建议先装淘宝镜像端，因为cnpm更快） 

    {{ snippets.install }}

该命令将在您的`package.json`文件中创建一个记录，并将该包安装到npm modules文件夹中。

例子

最简单的设置示例

首先需要将ng2-first-table指令导入到组件中。 

    {{ snippets.require }}

然后通过添加到您的模块的指令列表来注册它： 

    {{ snippets.directive }}

现在，我们需要配置表并将其添加到模板中。 组件开始工作的唯一必需设置是列配置。 我们在配置表中注册设置属性，并配置一些列 （设置文档）: 

    {{ snippets.settings }}

最后，我们将ng2-first-table组件放在模板中： 

    {{ snippets.template }}

在这一步，你将有一个最低配置的表，应该看起来像这样：
